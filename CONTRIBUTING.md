# Contributing

Cities That Care grows by districts: a real neighborhood, a fictional coordinator, six care projects.

- **To propose a district**: read [contribute.html](contribute.html) (or the live page) and write to nicolas.bronzina@gmail.com — or fill in the [district field template](docs/DISTRICT-TEMPLATE.md) directly.
- **To submit code or content**: follow the integration steps at the end of the template. The site is static (no build step); open `index.html` or run `python3 -m http.server` to preview.
- **Fiction boundary**: everything published joins a declared design fiction. Real streets, real climate pressures, fictional projects and people — see the [colophon](about.html).
